Detailed Description

Draws textured quads (sprites) into the current back buffer.

This is a 2d engine, so all geometry is a quad: a single unit quad lives in gpu memory and every draw_sprite call renders it with its own transformation, texture and tint color.

A frame looks like this:

renderer.begin_frame(clear_color);
renderer.draw_sprite(...); // as many times as you want
renderer.end_frame(); // executes, presents and synchronizes
