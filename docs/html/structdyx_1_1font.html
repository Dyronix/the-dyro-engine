Detailed Description

A fixed-grid bitmap font: one atlas texture holding every character in a regular grid, in ascii order.

All glyph data lives in the atlas image (content/fonts/), this struct only describes the grid so draw_text can find each character:

dyx::font font;
font.atlas = engine.get_texture_loader().load_from_file(path_to_atlas);
renderer.draw_text(font, "hello", { 20.0f, 20.0f }, 16.0f);
