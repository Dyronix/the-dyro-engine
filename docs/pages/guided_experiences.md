# Guided experiences {#page_guided_experiences}

These are hands-on exercises, not engine features. Each one walks you through
building a small system yourself on top of what the engine already gives you,
so you learn how the pieces fit together. The engine does not ship the result;
you write it as you follow along.

- @subpage page_camera_2d : build a scrolling 2D camera
- @subpage page_sprite_animation : build a sprite-animation helper
