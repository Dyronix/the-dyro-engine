# API reference {#mainpage}

The full class and function reference for BuzEngine, generated straight from
the engine headers so it never drifts from the code.

**Use the search box in the top-right corner.** Type the name of whatever you
are working with, such as `draw_sprite`, `was_key_pressed`, `noise_2d` or
`engine_settings`, and jump straight to its documentation. Every public class,
function and parameter is in here.

Or browse the class list in the sidebar. Start with buz::engine,
buz::renderer_2d and buz::input, the three classes your game talks to every
frame. From any class page, the **source browser** shows the full header.
