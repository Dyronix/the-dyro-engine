Get started → Browse the API
my_game.cpp
class my_game : public buz::game
{
    void initialize(buz::engine& e) override
    {
        m_ball = e.load_texture("textures/ball.png");
    }

    void draw(buz::engine& e, buz::renderer_2d& r) override
    {
        r.draw_sprite(*m_ball, { 640, 360 }, { 120, 120 });
    }

    std::shared_ptr<buz::texture> m_ball;
};

What's inside

A 2D renderer, keyboard and mouse input, texture and bitmap-font loading, and a set of small utilities, built on DirectX 12.

A class per system

The window, device, swap chain, renderer, input and texture loader are each their own class.

Built on DirectX 12

The renderer targets a modern graphics API directly, and runs on Windows 10 and 11 (64-bit).

Modern C++20

Concepts, ranges, std::span and std::format, the same standard current consoles compile with.

A searchable reference

Every public class, function and parameter is documented and searchable.

Guided experiences

Documentation on adding features to the engine, with pseudo code you can follow.

Almost no dependencies

No graphviz, no package manager, no heavy runtime. Generate the solution and press F5.
